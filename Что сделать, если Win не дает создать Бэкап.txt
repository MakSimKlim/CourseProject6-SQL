Зайти в Пуск -> Программы -> Microsoft SQL Server 2019 -> Средства настройки -> SQL Server Configuration Manager
В правом окне на SQL_Server_(MSSQLSERVER) правой кнопкой вызвать Свойства и скопировать Имя_учетной_записи
Зайти в папку, в которую будем делать Бэкап и правой кнопкой вызвать "Свойства"
В "Свойствах" перейти на вкладку "Безопасность" и нажать кнопку "Дополнительно"
В открывшемся окне выбрать "Добавить", затем в следующем открывшемся окне нажать "Выбрать субъект"
В открывшемся окне вставить в поле "Введите имена выбираемых объектов(примеры)" скопированное ранее Имя_учетной_записи, нажать ОК
Выбрать общие разрешения "Полный доступ", нажать Ок". Готово!

PS На всякий случай запустить "SQL Server Management Studio 19" от имени администратора
